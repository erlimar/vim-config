Erlimar Vim & NVim Config
=========================

My configurations and plugins for Vim and NVim.

## Setup Vim

- Create Vim config directory
```
$ mkdir -p ~/.vim
```

- Checkout GitHub config
```
$ cd ~/.vim
$ git clone https://github.com/erlimar/vim-config.git .
$ git checkout vim
```

- Install Plugins
```
:PlugInstall
```
